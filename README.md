## Dependências

1. Instalar o [Poetry](https://poetry.eustace.io/docs/#installation) no sistema;

2. Dentro do diretório do projeto, execute:

```
$ poetry lock
$ poetry install
```

Caso queira verificar se está tudo ok, execute:

`$ poetry shell`
